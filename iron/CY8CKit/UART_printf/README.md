# UART_printf
프로젝트를 열 때, 폴더 명을 `UART_printf.cydsn` 으로 변경해주세요.

## Hardware settings
* TX PIN=P1[5]
* RX PIN=P1[6]
* Heap Size=0x400

## UART settings
* Baud rate=115200
* Data bits=8
* Parity=None
* Stop bits=1

## Project description
1초마다 Hello World를 UART를 통해 출력함
